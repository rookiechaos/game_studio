# Sample Workspace Shape

This folder documents the expected structure of a generated project workspace.

Expected example path:

- `source_pack/workspaces/<project-id>/`

Expected artifact categories:

- intake artifacts
- plan artifacts
- run status and run history
- signoff record
- generated scene files
- delivery bundle outputs

Typical files:

- `frontend_intake_receipt.json`
- `frontend_plan_preview.json`
- `game_requirements.json`
- `game_plan.json`
- `scene_map.json`
- `release_checklist.json`
- `website_run_status.json`
- `website_run_history.json`
- `website_signoff.json`
- `website_run.log`
