# UI Package

This UI is a reference operator console for Japanese game workflow delivery teams.

Its main jobs are:

- collect intake data
- generate source-pack artifacts
- preview plan and delivery state
- export handoff markdown and delivery manifests
- optionally trigger the bundled runtime

## Important Defaults

- Default workspace root: `../workspaces`
- Default runtime mode: `artifact-only`
- Bundled runtime mode: `bundled-cli`
- Bundled runtime root: `../runtime/python`

## Included Source Areas

- `app/`
- `components/`
- `lib/`
- `scripts/`

## Commercial Boundary

This UI is included as source code.

It is not a hosted product, and production deployment is the buyer's responsibility unless separately contracted.
