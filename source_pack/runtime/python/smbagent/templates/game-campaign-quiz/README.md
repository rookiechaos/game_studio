# Game campaign quiz — scaffold

This is an unregistered scaffold for the future Japan game-studio product line.

It is intentionally not wired into the current SMB template registry yet.

## Intended deliverable shape

- `game/` — playable quiz logic
- `game/scenes/*.scene.json` — scene manifests for structural validation
- `site/` — campaign landing page
- `assets/` — project assets and placeholders
- `ops/` — reward, form, and campaign operation notes
- `analytics/` — event naming and instrumentation notes

## v1 constraints

- Mobile-first
- Japanese copy
- Lightweight browser deployment
- Campaign-grade rather than custom studio-grade
